Transfer Market Simulator

A C++ Object-Oriented Programming (OOP) project that simulates a football transfer market system.

Features

- Player registration system
- Club management system
- Transfer negotiation system
- Budget validation
- Transfer announcements
- Transfer history tracking
- Club squad dashboard
- Dynamic player transfers

Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Vectors
- Classes and Objects
- Modular Programming

System Components

Player Class
Stores:
- Name
- Age
- Position
- Market Value

Club Class
Stores:
- Club Name
- Budget
- Squad Players

Transfer System
Handles:
- Buying players
- Selling players
- Negotiations
- Budget updates

How To Run

1. Open the project in CodeBlocks
2. Build and Run the project
3. Use the menu system to interact with the simulator
